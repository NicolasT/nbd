An optional list of export names. If Nothing, export listing support will be disabled.

-> Pipe ByteString ByteString ByteString () m ExportName 

Run the new-style protocol negotiation handshake with a client The result will be the name of the export the client wants to connect to.

sendReplyData :: Monad m => Handle -> ByteString -> Pipe l i ByteString u m ()

Send a reply to the client indicating success for the given command handle. The provided data will be passed along (e.g. in response to a Read command).

sendReplyData' :: Monad m => Handle -> (ByteString -> m ()) -> m ()

Construct a reply to the client indicating success for the given handle. The reply bytes will be passed to the given action, which is responsible of pushing this data to the client, maybe followed by more data (e.g. when replying to a Read command).
